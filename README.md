## 使用方法
`./HisiLive` 加错误的信息会出现参数提醒。   
### 本地保存视频
```sh
./HisiLive -m file
```

### RTP协议发送
```sh
./HisiLive -m file -i 192.168.1.xxx 
```

VLC打开此目录下的play.sdp文件可以播放实时视频。
